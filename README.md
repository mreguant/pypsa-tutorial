# PyPSA Teaching Tutorial

This repository hosts the source code for a Jekyll-based documentation site focused on teaching **PyPSA**.

## 🌐 Live Site
The rendered version of this tutorial can be found at: 
**[https://mreguant.github.io/pypsa-tutorial/](https://mreguant.github.io/pypsa-tutorial/)**

## 🛠 Project Structure
- `_config.yml`: Site configuration (Just the Docs theme).
- `index.md`: The main landing page.
- `notebooks/`: Jupyter notebooks used for generating the teaching content.
- `assets/`: Static images and plots exported from PyPSA.

## 🚀 How to contribute
1. Add new lessons as Markdown files in the root or a subfolder.
2. Ensure images are exported as SVGs or high-res PNGs.
3. Update the `nav_order` in the front matter of new pages.
